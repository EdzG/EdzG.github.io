Projects

Things I've Built

Taiwan AQI Classification

Python scraper consuming the Ministry of Environment API to collect air quality index data across Taiwan. Benchmarked Random Forest, KNN, and LogitBoost classifiers for AQI category prediction, with feature engineering on temporal and meteorological variables.

Python scikit-learn Random Forest KNN LogitBoost

Wearable Spinal Posture Monitor

Wearable sensor system using ICM-20948 IMUs with Madgwick filter fusion to track spinal orientation in real time. I²C multiplexing allows concurrent multi-sensor readout. Accuracy validated against photogrammetric reference measurements.

ICM-20948 Madgwick Filter I²C Embedded C IMU Fusion

IoT Posture Detection Dashboard

End-to-end IoT capstone project: real-time posture classification running on an ESP32, serving live data and visual feedback through a self-hosted web dashboard over Wi-Fi. No external server required — the microcontroller acts as both sensor hub and web server.
